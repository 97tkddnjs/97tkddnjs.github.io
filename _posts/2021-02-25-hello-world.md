## 1. CPP로 Hello World 출력~

```cpp
#include<iostream>
using namespace std;
int main()
{
  cout<<"Hello World";
  return 0;
}

```

## 2. C로 Hello World 출력~

```c
#include<stdio.h>
int main()
{
  printf("Hello World");
  return 0;
}
```

## 2. python으로 Hello World 출력~

```python
print("Hello World")

```
